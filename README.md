# Tech Blog
## Description
A CMS-style blog where developers can publish their blog posts and comment on other developersâ€™ posts as well. This app follows the MVC paradigm in its architectural structure, using Handlebars.js as the templating language, Sequelize as the ORM, and the express-session npm package for authentication.
## Usage
Link - [Tech Blog](https://vast-fjord-09411.herokuapp.com/)
- API for all the users: [https://vast-fjord-09411.herokuapp.com/api/users](https://vast-fjord-09411.herokuapp.com/api/users)
